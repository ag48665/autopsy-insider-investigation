# Investigation Methodology

## Tools

- Autopsy 4.x
- Windows 11

## Steps

1. Create a new case.
2. Import the disk image.
3. Enable ingest modules.
4. Analyze recovered artifacts.
5. Review browser history.
6. Recover deleted files.
7. Build a timeline.
8. Generate a report.

## Objectives

- Recover deleted files
- Identify browser activity
- Detect USB devices
- Analyze user actions
- Collect digital evidence
